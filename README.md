## 📖 Overview

**EduNexus Global ERP** is a fully functioning, highly scalable school management application built purely with HTML, CSS, and Vanilla JavaScript. Designed around a **zero-dependency, local-first architecture**, it requires no cloud servers, APIs, or database installations. All persistent records are managed instantly within the browser's Sandbox.

It features a sophisticated **"Ivy League Modern" design system**, enforcing a premium academic aesthetic (Midnight Slate, Champagne Gold, Newsreader Serif) seamlessly synchronized across both the internal dashboards and public-facing portals.

## ✨ Core Modules & Features

- 🔐 **Role-Based Authorization System**
  - Instant toggle between strictly segmented interfaces for: `Administrator`, `Teacher`, `Student`, `Parent`, and `Librarian`.
- 📊 **Dynamic Analytics Dashboard**
  - KPI cards tracking enrollment, financial run-rate, attendance parity, and dynamic charts for quick operational insights.
- 🎓 **Academic & Pedagogy Control**
  - Manage students, teachers, active classes, subjects, and periods mapping.
  - Granular attendance tracking array.
  - Automated exam scoring architecture driving downloadable progressive PDF **Report Cards**.
- 📚 **Library & Asset Manager**
  - Full inventory pipeline tracking `Issued`, `Returned`, and `Overdue` book assets.
- 💵 **Financial & Fee Management**
  - Interactive ledgers detailing standard totals against partial or completed student commitments.
- 💾 **Advanced Portability Engine**
  - Run massive datasets strictly offline without latency.
  - Generate immediate `.json` structural backups of your enterprise.
  - Sophisticated restore logic supporting both **Non-Destructive Merge (Updates Only)** and **Strict Overwrite (Factory Wiping)** protocols.

## 🛠️ Technology Stack

1. **Frontend**: HTML5 Semantic Architecture
2. **Styling**: Vanilla CSS3 (Custom Properties/Tokens, Flexbox, Grid Layouts)
3. **Logic**: ECMAScript Modules (ES6+ Vanilla JavaScript)
4. **Database**: Browser native `localStorage` with JSON serialization overrides.

## 🚀 Getting Started

Because EduNexus operates completely client-side, getting the web app running takes literally three seconds.

1. **Clone or Download** the repository to your local machine.
2. Ensure all files (`index.html`, `landing.html`, `script.js`, `style.css`, and `landing.css`) are located within the identical root directory.
3. Open `index.html` in any modern web browser.
4. Use the Demo Login interface entirely offline.

### 🎭 Simulation Roles:
When testing the dashboard, select any scenario:
- **Admin**: Master rights enabling complete destruction controls and data backups.
- **Teacher**: Write-privileges limiting interaction to attendance matrices and grade arrays.
- **Parent/Student**: Hardlocked read-only modes masking global records and surfacing individual progress metrics.
